# cargo-doc-mcp

![ALPHA AS FUCK](https://img.shields.io/badge/version-ALPHA%20AS%20FUCK-ff69b4)

A lightweight, accurate MCP server for Rust documentation queries.

Provides AI assistants with structured access to Rust documentation through fuzzy searching of rustdoc JSON files. Unlike existing solutions that parse HTML or provide unfiltered results, cargo-doc-mcp delivers precise, token-efficient lookups of function signatures, type definitions, trait implementations, and source code.
